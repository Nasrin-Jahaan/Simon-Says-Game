Simon Says Game

An interactive memory-based game inspired by the classic Simon Says. This project is built using HTML, CSS, and JavaScript, and demonstrates core front-end development concepts such as DOM manipulation, event handling, and dynamic UI updates.

Overview

The Simon Says Game challenges the user to memorize and reproduce a sequence of colors. With each level, a new color is added to the sequence, increasing the difficulty progressively. The game ends when the user makes an incorrect selection.

Features

Dynamic level progression
Random sequence generation
Real-time score tracking
High score tracking
Visual feedback through animations
Game over detection with restart functionality

Tech Stack

HTML for structure
CSS for styling and animations
JavaScript for game logic and interactivity

How It Works

Press any key to start the game
A random color flashes on the screen
Repeat the sequence by clicking the buttons
Each level adds a new color to the sequence
The game ends when an incorrect input is given
